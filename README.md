# pfork
Daemonize given program. Detach from terminal.

## Building
Standard makefile. Here are some installation examples

    # compile and install
    make
    make install

    # compile with optimization
    make CFLAGS=-O2

    # you can set some variables. like PREFIX, BINDIR
    make PREFIX=/usr install
